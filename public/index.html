NeoTiny11

Windows 11 Gaming-Optimized Image Builder

Quick Install

Run this command in PowerShell (as Administrator):

irm https://neotiny11.vercel.app/install | iex

Features

No ISO Mounting

Just extract your Windows 11 ISO with 7-Zip and point to the folder. No mounting required!

Gaming Optimized

CPU mitigations disabled, GPU scheduling enabled, network latency optimized, and more.

Bloatware Free

Removes 50+ unnecessary apps while keeping gaming essentials like DirectX and VC++.

Privacy Focused

Disables telemetry, advertising, activity history, and other tracking features.

Hardware Bypass

Bypasses TPM, Secure Boot, RAM, and CPU requirements for older hardware.

Modular Scripts

Clean, readable PowerShell scripts that are easy to customize for your needs.

How to Use

Download Windows 11 ISO

Get the official ISO from Microsoft's website.

Extract the ISO

Use 7-Zip or WinRAR to extract the ISO contents to a folder.

Run the Builder

Open PowerShell as Administrator and run the install command above.

Select Source Folder

Choose the extracted ISO folder when prompted.

Get Your Optimized ISO

Find your gaming-optimized Windows 11 ISO in the output folder!

Important Notes

Always test on a virtual machine first. Some optimizations (like CPU mitigations) trade security for performance. Keep your original ISO as backup.
